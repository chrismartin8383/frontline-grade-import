Privacy Policy

Frontline Grade Import — Chrome Extension
Last updated: February 20, 2026

Summary: Frontline Grade Import does not collect, transmit, or share any personal data. All data processing occurs locally within your browser.

What the Extension Does

Frontline Grade Import reads student names, Local IDs, and grades from the Frontline Education ERP gradebook page and from grade files you provide (CSV or XLSX). This data is used solely to match students and fill grades into the gradebook on your screen.

Data Stored Locally

The extension stores two items in Chrome's built-in storage (chrome.storage):

  • QuizMaster URL — Your Google Apps Script web app URL, so you don't have to re-enter it each session.
  • Student email-to-ID mapping — A lookup table that links student email addresses to Frontline Local IDs, so you don't have to re-match students on every import.

This data stays entirely within your browser. It is not accessible to the developer or any third party.

Network Requests

The extension makes no external network requests except when you use the optional QuizMaster sync feature. In that case, the extension sends GET requests to a Google Apps Script web app URL that you provide and control. These requests fetch quiz lists and student grades from your own Apps Script deployment. No data is sent to any server operated by the extension developer.

Data Not Collected

The extension does not collect, transmit, or store:

  • Analytics or usage data
  • Browsing history
  • Cookies or tracking identifiers
  • Personal information beyond what is described above
  • Any data to external servers, databases, or third parties

Third-Party Services

The extension does not integrate with any third-party analytics, advertising, or data processing services.

Changes to This Policy

If this privacy policy is updated, the changes will be reflected on this page with an updated date.

Contact

If you have questions about this privacy policy, please reach out through the extension's Chrome Web Store support page.
